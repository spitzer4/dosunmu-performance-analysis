# Ayo Dosunmu Performance Analysis
## 1. Background and Overview
This project aims to analyze Ayo Dosunmu's 2023-24 performance as a key player for the Chicago Bulls.
The goal is to extract meaningful insights about his performance trends over the course of the season by analyzing game-by-game statistics and comparing these to the team's overall performance.

This project uses historical game data, performance metrics, and statistical analysis to provide an overview of Ayo’s contributions to the team's success. The key focus areas are:

- **Player performance metrics:** Points, assists, rebounds, shooting efficiency, and more.
- **Team impact:** How Ayo’s performance correlates with the team’s success (wins vs. losses).
